Introduction

This is the Laravel CRUD API Documentation

Base URL: https://laravel-crud-api-production-223e.up.railway.app

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Todos

Get all todos.

Retrieves a list of todos with optional ordering and search parameters.

Example request:
curl --request GET \
    --get "https://laravel-crud-api-production-223e.up.railway.app/api/todos?order=%22desc%22.&search=%22meeting%22." \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "https://laravel-crud-api-production-223e.up.railway.app/api/todos"
);

const params = {
    "order": ""desc".",
    "search": ""meeting".",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Example response (200):


{
    "status_code": 200,
    "todos": []
}

Example response (422):

    "status_code": 422,
    "message": "Order type can only be asc or desc"

Create a new todo.

Adds a new todo to the database.

Example request:
curl --request POST \
    "https://laravel-crud-api-production-223e.up.railway.app/api/todos/new" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"title\": \"\\\"Finish project\\\".\",
    \"details\": \"\\\"Complete all tasks for the project\\\".\",
    \"status\": 0
}"

const url = new URL(
    "https://laravel-crud-api-production-223e.up.railway.app/api/todos/new"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "title": "\"Finish project\".",
    "details": "\"Complete all tasks for the project\".",
    "status": 0
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

{
    "status_code": 201,
    "message": "Todo created successfully"
}
